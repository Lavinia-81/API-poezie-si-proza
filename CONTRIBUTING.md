# Contributing Guidelines

Thank you for your interest in contributing to the Romanian Classical Literature API!

## How to Contribute

### 1. Fork the repository
Create your own copy of the project.

### 2. Create a feature branch
Use a descriptive name:
`git checkout -b feature/add-new-author`

### 3. Follow the project structure
All authors must follow the existing folder format:
- poezii/
- proza/
- bibliografie/
- author.json

### 4. Code style
- Use clear, modular functions
- Avoid hard‑coding paths
- Keep endpoints consistent

### 5. Submit a Pull Request
Include:
- description of changes
- screenshots if relevant
- reference to issues

## Reporting Issues
Use GitHub Issues and include:
- steps to reproduce
- expected behavior
- actual behavior
- logs if available

Thank you for helping preserve Romanian literary heritage!
