Politica de Confidențialitate

Ultima actualizare: 2026

1. Introducere

Această Politică de Confidențialitate explică modul în care Poems & Prose API – Romanian Classical Literature colectează, utilizează și protejează datele utilizatorilor.

2. Date colectate

  • Adresa de email (pentru autentificare și acces la Dashboard)
  • Detalii despre abonament (plan activ, limite de utilizare)
  • Statistici de utilizare API (număr de request‑uri, endpoint‑uri accesate)

3. Cum folosim datele

Datele sunt utilizate exclusiv pentru:

  • gestionarea contului și a abonamentului
  • generarea cheii API
  • monitorizarea limitelor de utilizare
  • notificări legate de cont

4. Securitate

Datele sunt stocate în medii securizate, iar conexiunile sunt protejate prin HTTPS.

5. Drepturile utilizatorului

Utilizatorii pot accesa și vizualiza datele asociate contului lor (email, plan activ, statistici de utilizare). Deoarece API‑ul este strict read‑only, utilizatorii nu pot modifica, edita sau șterge conținutul literar furnizat prin API.

Conținutul disponibil prin acest API provine exclusiv din opere aflate în domeniul public. Deși patrimoniul literar românesc cuprinde peste 300 de autori canonici, multe opere sunt încă protejate de drepturi de autor și nu pot fi distribuite sau modificate printr-o interfață publică. API‑ul oferă acces doar la opere care pot fi legal conservate, indexate și reutilizate în scop educațional.

6. Actualizări

Politica poate fi actualizată periodic. Versiunea curentă este afișată pe această pagină.

Privacy Policy

Last updated: 2026

1. Introduction

This Privacy Policy explains how the Poems & Prose API – Romanian Classical Literature collects, uses, and protects user data.

2. Data we collect

  • Email address (for authentication and Dashboard access)
  • Subscription details (active plan, usage limits)
  • API usage statistics (request count, accessed endpoints)

3. How we use the data

Data is used exclusively for:

  • account and subscription management
  • API key generation
  • usage monitoring
  • account‑related notifications

4. Security

All data is stored securely, and all connections are protected via HTTPS.

5. User rights

Users may access and view the data associated with their account (email, active plan, usage statistics). Since the API is strictly read‑only, users cannot modify, edit, or delete any literary content provided through the API.

The content available through this API is sourced exclusively from works in the public domain. Although Romanian literary heritage includes over 300 canonical authors, many works remain under copyright protection and cannot be distributed or modified through a public interface. The API provides access only to works that can be legally preserved, indexed, and reused for educational purposes.

6. Updates

This policy may be updated periodically. The current version is displayed on this page.
